## ⚡ Key Architecture & Features

### 👤 SEO & Discoverability
Built with highly optimized custom OpenGraph, title, and meta tag strings allowing the portfolio to rank easily on search engine indexes for relevant engineering targets.

### 🌓 Theme Persistence
Includes an inline, blocking JavaScript script inside the `<head>` to immediately access `localStorage` and check native user preferences (`prefers-color-scheme`). This prevents flashes of unstyled content (FOUC) when switching dark/light presentation styles.

### 📬 Serverless Contact Form
Leverages a serverless design utilizing the FormSubmit API to securely route prospective recruiter queries directly to my inbox without exposing back-end validation credentials.
